Achtung! Blinkenlights

What does it do?

Displays a (square) grid of clickable cells, with default width/height of 50 cells. Clicking a cell increments the value of that cell, as well as the value of all the cells in the same row and column. After each increment, a cell briefly lights up in yellow. If two neighbouring cells (up, down, left, right) contain a value of four and two, both these cells light up briefly in green, and then reset their values to zero.
